# Smoodies Website 🎸

Este projeto consiste no desenvolvimento do website oficial da banda **Smoodies**. O site foi criado como parte prática da Unidade de Competência **UC604 - HTML/CSS** do curso **CET de Programação**.

O objetivo principal é apresentar a identidade visual da banda, agenda de concertos, e contactos, aplicando boas práticas de marcação semântica e estilização moderna.

## 🚀 Tecnologias Utilizadas

*   **HTML5:** Estruturação semântica de conteúdos.
*   **CSS3:** Estilização responsiva e animações.
*   **Git & GitHub:** Controlo de versões e alojamento via GitHub Pages.

## 🛠️ Funcionalidades

- [x] Design responsivo (adaptável a telemóveis e tablets).
- [x] Secção de Biografia da banda.
- [x] Galeria de fotos e vídeos.
- [x] Formulário de contacto para booking.
- [x] Integração com redes sociais (Instagram).
